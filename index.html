Numerical Integration

Composite
Trapezoidal Rule

A fundamental technique that approximates a definite integral by partitioning the interval into subintervals and summing trapezoid areas beneath the curve.

Numerical Integration O(h²) convergence Closed Newton–Cotes n+1 evaluations
01

Mathematical Discussion

The Composite Trapezoidal Rule estimates the definite integral $\int_a^b f(x)\,dx$ by subdividing $[a,b]$ into $n$ equal subintervals of width $h = \tfrac{b-a}{n}$, then approximating the area under each subinterval as a trapezoid.

Master Formula
$$\int_a^b f(x)\,dx \;\approx\; \frac{h}{2}\Bigl[f(x_0) + 2\sum_{i=1}^{n-1}f(x_i) + f(x_n)\Bigr]$$

Here $x_i = a + i\,h$ for $i = 0, 1, \ldots, n$ and $h = \dfrac{b-a}{n}$. The endpoints $f(x_0)$ and $f(x_n)$ appear with coefficient 1, while all interior nodes carry coefficient 2.

Step Size
$$h = \frac{b - a}{n}$$
Truncation Error Bound
$$|E_T| \leq \frac{(b-a)^3}{12n^2}\,\max_\xi|f''(\xi)|$$

The error of the composite rule is $O(h^2)$: doubling $n$ reduces error by a factor of four. Continuous functions are guaranteed to converge as $n \to \infty$.

2nd-order accuracy in h n + 1 function evaluations Riemann-integrable convergence Best for smooth functions
