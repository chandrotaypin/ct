Numerical Integration

Composite
Trapezoidal Rule

A fundamental technique for numerical integration that approximates a definite integral by partitioning the interval into subintervals and applying the trapezoidal rule to each.

Numerical Integration O(h²) Local Error O(h⁴) Composite Error Closed Newton–Cotes
∫
Mathematical Discussion Theory & Formula

The Composite Trapezoidal Rule estimates the definite integral $\int_a^b f(x)\,dx$ by subdividing $[a,b]$ into $n$ equal subintervals of width $h = \tfrac{b-a}{n}$, then approximating the area under each subinterval as a trapezoid.

Master Formula
$$\int_a^b f(x)\,dx \;\approx\; \frac{h}{2}\Bigl[f(x_0) + 2\sum_{i=1}^{n-1}f(x_i) + f(x_n)\Bigr]$$

where $x_i = a + i\,h$ for $i = 0, 1, \ldots, n$ and $h = \dfrac{b-a}{n}$. The endpoints $f(x_0)$ and $f(x_n)$ appear with coefficient 1, while all interior nodes $f(x_1), \ldots, f(x_{n-1})$ appear with coefficient 2.

Step Size
$$h = \frac{b - a}{n}$$
Truncation Error Bound
$$|E_T| \leq \frac{(b-a)^3}{12n^2}\,\max_{a \le \xi \le b}|f''(\xi)|$$

The error of the composite rule is $O(h^2)$: doubling the number of subintervals $n$ reduces the error by a factor of four. This makes the method both simple to implement and convergent for any integrable function — continuous functions are guaranteed to converge as $n \to \infty$.

Order of accuracy: 2nd order in $h$ Function evaluations: $n + 1$ Converges for any Riemann-integrable $f$ Best for smooth functions
